gesture
=======

触屏手势库，基于Winter的第一版[Gesture](https://github.com/SNDA-FE/Gesture)和zepto.js完成。原版Gesture与zepto在事件绑定和事件冒泡的使用上不太好协作，zepto.js自身的touch很粗糙，没有gesture的事件丰富。所以结合两者的优势，重写了一版。

## 手势事件
- tap
- doubletap
- press
- pressend
- flick
- pan
- panstart
- panend
- dualtouch
- dualtouchstart
